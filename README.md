# Job Posting Web Application

## Description

This Laravel 11 + Vue 3 application is designed to facilitate job postings for employers and job searching for employees. It provides a platform for businesses to advertise open positions and for individuals to find suitable employment opportunities.

## Features

- **Job Postings**: Employers can post job listings, including job title, description, requirements, and application details.
- **Job Search**: Job seekers can search for jobs by various criteria such as location, job type, and industry.
- **User Authentication**: Secure login and registration for both employers and job seekers.
- **Profile Management**: Users can manage their profiles, including updating personal information and tracking job applications.
- **Responsive Design**: The application is optimized for both desktop and mobile devices.

## Installation

### Prerequisites

- PHP 8.1 or higher
- Composer
- Node.js and npm/yarn
- MySQL or another supported database
